# Student Management System

This is a **Java-based Student Management System** project developed in Eclipse.  
It includes source code, Eclipse project configuration, and resources.

## Features
- Add, update, and delete student records
- Search and view student information
- Simple console-based interface (or GUI if included)

## Setup
1. Clone this repository or download the files.
2. Open in **Eclipse IDE**.
3. Run the `main` class to start the application.
